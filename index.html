Welcome to what I call JayOS

This project uses GNU makefiles and Bash shell scripts to build a complete Linux distribution. All of its components are built from source. The result is a turnkey solution to creating a compact, highly-customizable operating system.

JayOS includes a lot of standard network security and filesystem tools, along with a full-blown development environment with many programming tools and libraries including gcc, gdb, ddd, perl, gtk, tcl/tk, ruby, python, qt3/4, and glade-3. See the package list for details.

A small footprint lets it run well on commodity hardware, but it can be configured to use extra RAM to run entirely from memory. And for the security-conscious among us, it offers in-RAM filesystem encryption with plausible deniability to provide a more secure mobile workstation.

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ mkdir -p /lfs/build && cd /lfs/build
$ git clone git://github.com/kharmajbird/JayOS

If you have a Mac Mini (my development platform of choice :) grab two 1 gigabyte USB thumbdrives and follow these instructions to try out JayOS without disturbing your OS X drive.

Otherwise, I recommend downloading the latest ISO and either burning it to CDROM, or booting it under QEMU emulation.

Development

If you want to build your own ISO image from source, (see why) you will need LFS, BLFS, and various other tarballs. So first clone the git repo as described above. Then download the tarballs all at once with wget by running dl.sh 

$ /lfs/build/JayOS/scripts/dl.sh

Then to initiate the build, run build.sh 
$ /lfs/build/JayOS/scripts/build.sh
This works best when launched from the shell prompt of the Live CD. Building from other Linux distributions has yet to be tested, so YMMV.
